Cómo descargar tu partida automáticamente:
1. Pulsa el botón azul de arriba
2. Se abre el juego en una pestaña nueva
3. Pulsa F12 → pestaña Console
4. Copia y pega este código exactamente y pulsa Enter:
indexedDB.open('infinite-craft').onsuccess = e => {
  const db = e.target.result;
  const tx = db.transaction('items', 'readonly');
  const store = tx.objectStore('items');
  store.getAll().onsuccess = e2 => {
    const data = e2.target.result;
    const blob = new Blob([JSON.stringify(data)], {type: 'application/json'});
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'mi_progreso_infinite_craft.json';
    a.click();
    console.log('✅ Archivo descargado');
  };
};
5. Se descargará automáticamente el JSON
6. Vuelve aquí y súbelo con el primer botón
